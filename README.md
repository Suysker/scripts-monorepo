# JS Scripts Monorepo 🚀

> **一个仓库管理多套浏览器脚本：视频快捷控制、流媒体缓冲增强、YFSP 页面解锁增强。**

![License](https://img.shields.io/github/license/Suysker/scripts-monorepo?style=flat-square) ![Repo](https://img.shields.io/badge/Repo-Monorepo-blue?style=flat-square) ![Language](https://img.shields.io/badge/Language-JavaScript-yellow?style=flat-square)

GitHub：`https://github.com/Suysker/scripts-monorepo`

---

## 📦 包含项目

### 1) 黄金左右键（Golden-Left-Right）

- 目录：[`Golden-Left-Right/`](./Golden-Left-Right/)
- 脚本：`Golden-Left-Right/Golden-Left-Right.js`
- 作用：按左右键实现视频加速/减速/快进，适合追剧和跳片段。

### 2) 流媒体加速缓冲（StreamBoost）

- 目录：[`StreamBoost/`](./StreamBoost/)
- 脚本：`StreamBoost/StreamBoost.js`
- 作用：增强 HLS 场景下的缓冲与预取，降低卡顿和反复缓冲。

### 3) YFSP Unlocker

- 目录：[`yfsp/`](./yfsp/)
- 脚本：`yfsp/yfsp-unlocker.js`
- 作用：针对 YFSP 站点解锁清晰度/倍速/弹幕样式前端 UI，并清理常见广告遮罩。
